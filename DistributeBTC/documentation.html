Distribute BTC

To create a new distributer you will need to access the admin panel. This section is only available to the administrator of the app and not visible to the public.

Click the “New distributer” button and fill in the form. When you are done, click the “Save” button, it's that simple!

Parameters:
Distributer
Name Required Type Description
Status yes "Pending", "Active" or "Disabled" When a new distributer is created, it will have the “Pending” status. Once you are ready, set this to “Active”. If you want to temporarily disable a distributer, set this to “Disabled”.
AddressType yes "BIP44" or "Private key" Select which type of address you will use for this distributer, an address from the built-in BIP44 HOT wallet or an address derived from a private key.
Wallet index for type "BIP44" integer If you chose “BIP44” for the addressType, you can specify an index number for the address to be used, default value is the current highest indexnumber +1. This indexnumber must be unique across all distributers.
Private key for type "Private key" string If you chose “Private key” for the addressType, you need to enter the private key here.
Name yes string Give the distributer a short descriptive name.
Description yes string In this area you can enter a more detailed description of the distributer and its purpose.
Distribution source yes "Custom distribution", "SIL", "LBL", "LRL" or "LSL" Select the source of the distribution, It can be a dynamic distribution according to a SIL or one of the other Linked Lists or a fixed custom distribution.
Custom distribution no JSON It is possible to specify a custom distribution, but must be in a special format: a JSON list of address-value pairs, just like a SIL, be careful when setting this manually!!
Registration address yes string A Bitcoin address that will be used to generate the SIL for the distribution.
Registration blockheight yes integer The blockheight for the registration. Set this to 0 to use the most recent blockheight.
Registration XPUB yes string The extended public key used to generate the Linked Lists.
Creator yes string The name of the person responsible for the distributer.
Creator email yes email address An email address to contact the person responsible for the distributer.
Visibility yes "Public" or "Private" Public distributers are shown on the main page. Private distributers are only accessible to those who have a link containing the id of the distributer.
Threshold amount yes integer Specify a minimum amount to trigger the distribution. As long as the balance of address is below this value, nothing will happen.
Minimum amount yes integer The minimum output value for each recipient of the distribution, if a recipients proportional share is below this value, it is excluded from that particular distribution.
Youtube no string Here you can enter the video code of a youtube video. This is the code that comes after the “watch?v=” part in the youtube url.
Distribution fee percent no float Optionally a fee can be deducted from the distributed amount.
Distribution fee address no string Specify an address to send the distribution fees to.

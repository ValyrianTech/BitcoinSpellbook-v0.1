Blockchaindata: transactions

Returns a JSON-object containing all transactions received by a specified Bitcoin address

Arguments:
Name Required Type Description
format yes string Format of the results, should be “json”
address yes string A Bitcoin address
Example: transactions?format=json&address=1NC8LqAB99bYM9wVoD2grdYMMZAhjwy57A
Return values:
Name Type Description
success integer 1 = success, 0 = fail
TXS array list of all transactions
- txid string Transaction hash
- blockheight integer Block in which this transaction is included.
- confirmations integer Number of confirmations the transaction has.
- inputs array list of all inputs
- - address string Address of the input
- - value integer Value of the input
- outputs array list of all outputs
- - address string Address of the output
- - value integer Value of the output
- - spent boolean Has the output been spent?
- primeInputAddress string the first inputAddress sorted alphabetically
- receivingValue integer The value that was received in this transaction
- receiving boolean Was the transaction received by or sent by this address?
provider string 3rd party provider that supplied the data.
error string error message in case something went wrong
Return example: 
{
    "success": 1,
    "TXS":[{u'blockheight': 375786,
              u'confirmations': 32673,
              u'inputs': [{u'address': u'1Robbk6PuJst6ot6ay2DcVugv8nxfJh5y',
                           u'value': 220125}],
              u'outputs': [{u'address': u'1Robbk6PuJst6ot6ay2DcVugv8nxfJh5y',
                            u'spent': False,
                            u'value': 110716},
                           {u'address': u'1NC8LqAB99bYM9wVoD2grdYMMZAhjwy57A',
                            u'spent': True,
                            u'value': 100000}],
              u'primeInputAddress': u'1Robbk6PuJst6ot6ay2DcVugv8nxfJh5y',
              u'receivedValue': 100000,
              u'receiving': True,
              u'txid': u'5aeec17e3ff00542240bd7b47188c396342b6bebd2351e500d561b70b21c2eb8'},
...
]}
Blockchaindata: utxos

Returns a JSON-object containing all unspent transaction outputs for a list of Bitcoin addresses. Multiple addresses seperated by "|".

Arguments:
Name Required Type Description
format yes string Format of the results, should be “json”
addresses yes string A list of Bitcoin addresses, separated by "|".
Example: utxos?format=json&addresses=1Robbk6PuJst6ot6ay2DcVugv8nxfJh5y|1SansacmMr38bdzGkzruDVajEsZuiZHx9
Return values:
Name Type Description
success integer 1 = success, 0 = fail
UTXOs array list of all unspent transaction outputs
- confirmations integer Number of confirmations of the unspent transaction output
- output string format: TransactionId:index
- address string Address of the unspent transaction output
- value integer Value of the unspent transaction output
- block_height string Blockheight of the unspent transaction output.
provider string 3rd party provider that supplied the data.
error string error message in case something went wrong
Return example: 
{
    "success": 1,
    "UTXOs": [{u'address': u'1Robbk6PuJst6ot6ay2DcVugv8nxfJh5y',
              u'block_height': 371357,
              u'confirmations': 37103,
              u'output': u'39bb5f5d50882227f93b980df15ea676414f0363770a0174a13c8f55c877b598:1',
              u'value': 5000},
             {u'address': u'1Robbk6PuJst6ot6ay2DcVugv8nxfJh5y',
              u'block_height': 375786,
              u'confirmations': 32674,
              u'output': u'5aeec17e3ff00542240bd7b47188c396342b6bebd2351e500d561b70b21c2eb8:0',
              u'value': 110716},
              ...
    ],
    "provider": "Blocktrail.com"
}
Blockchaindata: balances

Returns a JSON-object containing the balances for a list of Bitcoin addresses. Multiple addresses seperated by "|".

Arguments:
Name Required Type Description
format yes string Format of the results, should be “json”
addresses yes string A list of Bitcoin address, separated by "|".
Example: balances?format=json&addresses=1Robbk6PuJst6ot6ay2DcVugv8nxfJh5y|1SansacmMr38bdzGkzruDVajEsZuiZHx9
Return values:
Name Type Description
success integer 1 = success, 0 = fail
balances dict Dictionary containing all balances of the supplied addresses
- balance integer Balance of the address in Satoshis
- received integer Total received value of the address in Satoshis
- sent integer Total sent value of the address in Satoshis
provider string 3rd party provider that supplied the data.
error string error message in case something went wrong
Return example: 
{
    "success": 1,
    "balances": {u'1Robbk6PuJst6ot6ay2DcVugv8nxfJh5y': {u'balance': 389493,
                                                        u'received': 1534705,
                                                        u'sent': 1145212},
                u'1SansacmMr38bdzGkzruDVajEsZuiZHx9': {u'balance': 626643,
                                                        u'received': 1325281,
                                                        u'sent': 698638}},
    "provider": "Blocktrail.com",
}
Blockchaindata: primeInputAddress

Returns a JSON-object containing the primeInputAddress of a transaction. This is the first of the input addresses sorted alphabetically.

Arguments:
Name Required Type Description
format yes string Format of the results, should be “json”
txid yes string A transaction hash
Example: primeInputAddress?format=json&txid=39bb5f5d50882227f93b980df15ea676414f0363770a0174a13c8f55c877b598
Return values:
Name Type Description
success integer 1 = success, 0 = fail
PrimeInputAddress string Prime Input Address of the transaction
provider string 3rd party provider that supplied the data.
error string error message in case something went wrong
Return example: 
{
    "success": 1,
    "PrimeInputAddress": "1ETPtznVTgowgGdfqREtDtmA6JXteBxFEL",
    "provider": "Blocktrail.com"
}
Blockchaindata: latestBlock

Returns a JSON-object containing the details of the latest Bitcoin block.

Arguments:
Name Required Type Description
format yes string Format of the results, should be “json”
Example: latestBlock?format=json
Return values:
Name Type Description
success integer 1 = success, 0 = fail
latestBlock dict dictionary containing the details of the latest block
- merkleroot string merkleroot of all transactions in this block
- time integer timestamp of this block
- hash string Block hash
- size integer size of this block
- height integer Height of this block
error string error message in case something went wrong
Return example: 
{
    "success": 1,
    "latestBlock": {
        "merkleroot": "c89cc94dca93356013490802b3be296b01d9861f525595841f29632f94dc8875",
        "time": 1453939350,
        "hash": "0000000000000000048cdd145fac0aa26c16b3071c47747bd9eaf34cc3389708",
        "size": 949137,
        "height": 395352
    },
    "provider": "Blocktrail.com",
}

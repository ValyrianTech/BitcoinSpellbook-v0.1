HDForwarder

To create a new forwarder you will need to access the admin panel. This section is only available to the administrator of the app and not visible to the public.

Click the “New forwarder” button and fill in the form. When you are done, click the “Save” button, it's that simple!

Parameters:
HD Forwarder
Name Required Type Description
Status yes "Pending", "Active" or "Disabled" When a new forwarder is created, it will have the “Pending” status. Once you are ready, set this to “Active”. If you want to temporarily disable a forwarder, set this to “Disabled”.
AddressType yes "BIP44" or "Private key" Select which type of address you will use for this forwarder, an address from the built-in BIP44 HOT wallet or an address derived from a private key.
Wallet index for type "BIP44" integer If you chose “BIP44” for the addressType, you can specify an index number for the address to be used, default value is the current highest indexnumber +1. This indexnumber must be unique across all forwarders.
Private key for type "Private key" string If you chose “Private key” for the addressType, you need to enter the private key here.
Name yes string Give the forwarder a short descriptive name.
Description yes string In this area you can enter a more detailed description of the forwarder and its purpose.
XPUB yes string Specify the extended public key of a BIP44 wallet, this is where the bitcoins will be forwarded to.
Creator yes string The name of the person responsible for the forwarder.
Creator email yes email address An email address to contact the person responsible for the forwarder.
Visibility yes "Public" or "Private" Public forwarders are shown on the main page. Private forwarders are only accessible to those who have a link containing the id of the forwarder.
Minimum amount yes integer Specify a minimum amount (in Satoshis) to trigger the forwarder. If a transaction is received with a value below this amount, the bitcoins will be returned to the sender instead of forwarded.
Youtube no string Here you can enter the video code of a youtube video. This is the code that comes after the “watch?v=” part in the youtube url.
Forwarding fee percent no float Optionally a fee can be deducted from the forwarded amount.
Forwarding fee address no string Specify an address to send the forwarding fees to.
Confirmation amount no integer Optionally specify an amount in Satoshis to send back to the sender as a sign of confirmation.

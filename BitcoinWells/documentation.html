Bitcoin Wells

To create a new well you will need to access the admin panel. This section is only available to the administrator of the app and not visible to the public.

Click the “New well” button and fill in the form. When you are done, click the “Save” button, it's that simple!

Optionally each well can have multiple campaigns and each campaign can have multiple milestones.

Parameters:
Well
Name Required Type Description
Name yes string A short descriptive name for the well.
Bitcoin address yes string A Bitcoin address for the well, this is where people will send bitcoins to.
Status yes "Pending", "Active" or "Disabled" When a new well is created, it will have the “Pending” status. Once you are ready, set this to “Active”. If you want to temporarily disable a well, set this to “Disabled”.
Visibility yes "Public" or "Private" Public wells are shown on the main page. Private wells are only accessible to those who have a link containing the id of the well.
Description yes string In this area you can enter a more detailed description of the well and its purpose.
Tags no string Enter a few short keywords to label the well, separated with “;”
Youtube no string Here you can enter the video code of a youtube video. This is the code that comes after the “watch?v=” part in the youtube url.
Creator yes string The name of the person responsible for the well.
Creator email yes email address An email address to contact the person responsible for the well.
Sort order yes integer Set the order to display the well on the main page.
Campaign
Name Required Type Description
Name yes string A short descriptive name for the campaign.
Description yes string A more detailed description of the campaign.
Target amount yes integer Specify the target amount in Satoshis for this campaign.
Youtube no string Each campaign can show a separate youtube video.
Milestone
Name Required Type Description
Name yes string A short descriptive name for the milestone.
Description yes string A more detailed description of the milestone.
Percent yes float Specify where the milestone is in the campaign. This must be a percentage and will automatically be calculated based on the parent campaigns target.
Action yes "None", "Reveal text", "Reveal link", "Send email" Choose the action to occur when the milestone is achieved.
Reveal text no string Enter the text to reveal only when the milestone has been reached.
Reveal link text no string Enter the text to display in the link.
Reveal link URL no string Enter the url to reveal only when the milestone has been reached.
Mail to no string Enter the recipient’s email address
Subject no string Enter a subject for the email.
Body no string Enter a body for the email.
